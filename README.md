Zeillogs
--------

A nice way to present stories about sailing.

http://jieter.github.io/saillogs/

## Start server
After `npm install` run `grunt`. For the read-only case, every static webserver should work.

## grunt tasks

```
grunt import-sailplanner:<key>
```
